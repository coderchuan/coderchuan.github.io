## 魔术方法
* `public __sleep ( void ) : array`
    * 功能：使用[serialize()](https://www.php.net/manual/zh/function.serialize.php)序列化对象实例时，定义要被序列化的属性值。未被返回的属性名称的值不会被序列化
* `public __wakeup ( void ) : void`
    * 功能：使用[unserialize()](https://www.php.net/manual/zh/function.unserialize.php)反序列化对象实例时，对被反序列化的对象实例进行初始化。例如重建数据库连接
* `public __toString ( void ) : string`
    * 功能：对象实例的字符串内容
