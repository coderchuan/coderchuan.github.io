## 滚动条重绘
```css
/**********滚动条重绘 BEGIN**********/
.scroll_bar_x{
    overflow-y: hidden;
    color: #000;
    font-size: 16px;
}
/* 滚动条整体 */
.scroll_bar_x::-webkit-scrollbar {
    height:10px;/*滑轨宽度*/
}
/* 滑块样式 */
.scroll_bar_x::-webkit-scrollbar-thumb {
    border-radius:100px;
    background:#C1C1C1;/*滑块颜色*/
}
/* 轨道样式 */
.scroll_bar_x::-webkit-scrollbar-track {
    border-radius:100px;
    background:rgba(0,0,0,0.1);
}
/**********滚动条重绘 END**********/




/**********滚动条重绘 BEGIN**********/
.scroll_bar_y{
    overflow-x: hidden;
    color: #000;
    font-size: 16px;
}
/* 滚动条整体 */
.scroll_bar_y::-webkit-scrollbar {
    width:4px;/*滑轨宽度*/
}
/* 滑块样式 */
.scroll_bar_y::-webkit-scrollbar-thumb {
    border-radius:2px;
    background:#C1C1C1;/*滑块颜色*/
}
/* 轨道样式 */
.scroll_bar_y::-webkit-scrollbar-track {
    border-radius:2px;
    background:rgba(0,0,0,0.1);
}
/**********滚动条重绘 END**********/
```
