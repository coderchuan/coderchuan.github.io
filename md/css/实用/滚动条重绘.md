## 滚动条重绘
```css
/**********滚动条重绘 BEGIN**********/
.scroll_bar{
    overflow-x: auto;
    overflow-y: auto;
}
/* 滚动条整体 */
.scroll_bar::-webkit-scrollbar {
    width:8px;/*y滑轨宽度*/
    height:8px;/*x滑轨宽度*/
}
/* 滑块样式 */
.scroll_bar::-webkit-scrollbar-thumb {
    border-radius:50px;
    background:#C1C1C1;/*滑块颜色*/
}
/* 轨道样式 */
.scroll_bar::-webkit-scrollbar-track {
    border-radius:50px;
    background:rgba(0,0,0,0);/*滑轨颜色*/
}
/**********滚动条重绘 END**********/
```
