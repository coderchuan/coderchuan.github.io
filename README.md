## 使用方法
1. 在`note`文件夹中添加后缀为`.md`的文件,`note`文件夹下可再创建文件夹
2. 运行`php auto_index.php`，将会自动为`note`文件夹下的`.md`文档创建主页索引，并复制note目录下的index.md到仓库主目录下。php版本要求为`7.2`及以上，低于`7.2`版本未测试
